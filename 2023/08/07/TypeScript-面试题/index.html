✅ 什么是 TypeScript？

TypeScript 是 JavaScript 的超集，作用是为 JavaScript 提供静态类型系统。

✅ TypeScript 的优缺点是什么？

优点：

  • 静态类型
  • 前置错误
  • 易于重构

缺点：

  • 学习成本
  • 开发成本
  • 编译成本

✅ TypeScript 有哪些特性？

  • 接口
  • 别名
  • 类型注解
  • 泛型
  • 枚举
  • 断言
  • 命名空间
  • 声明文件
  • 工具函数
  • keyof

✅ interface 和 type 的区别是什么？

合并声明：

  • interface可以合并声明，多个同名的接口声明会自动合并成一个接口。
  • type不支持合并声明，多个同名的类型声明会报错。

可扩展性：

  • interface可以通过 extends 关键字实现继承。
  • type可以通过 & 关键字实现交叉类型。

类型描述：

  • interface适合描述对象类型。
  • type适合描述复杂类型，比如枚举类型、联合类型、交叉类型等。
